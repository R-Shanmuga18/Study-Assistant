## Diagram Format

All diagrams use minimal ASCII art for maximum compatibility and readability across all platforms and editors.

## How to View

All ASCII diagrams display correctly in:
1. **GitHub**: Renders directly in markdown files
2. **VS Code**: View in any text editor
3. **Terminal**: Cat or less commands work perfectly
4. **Any text viewer**: No special extensions needed
